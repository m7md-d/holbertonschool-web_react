# holbertonschool-web_react

Holberton School web/React curriculum. Each top-level directory is a self-contained
module with its own toolchain and task series.

## Modules
| Module | What it covers |
|--------|----------------|
| [TypeScript](./TypeScript) | TypeScript fundamentals — interfaces, classes, advanced types, type guards, ambient/declaration namespaces, and nominal typing — bundled with webpack + ts-loader |
